# Bookshop

Невеликий консольний проєкт книжкового магазину, побудований на модульній архітектурі. Складається з декількох незалежних підсистем: користувачі, перегляд книг, робота з кошиком та картка товару.

## Вимоги
- `g++` з підтримкою **C++17** або новішого стандарту
- Linux / macOS / WSL (Windows також можна, якщо є MinGW або clang)

## Структура проєкту
```
bookshop/
├── src/
│   ├── main.cpp
│   ├── user/
│   │   └── user.cpp
│   ├── showBooks/
│   │   └── showBooks.cpp
│   ├── productcard/
│   │   └── productcard.cpp
│   └── Basket/
│       └── basket.cpp
└── include/ (опціонально, якщо буде виділення .h окремо)
```

## How to compile
Для компіляції необхідно зібрати **всі .cpp** файли включно з підпапками:
```
g++ -std=c++17 -g $(find src -name '*.cpp') -o bookshop
```
Ця команда автоматично знаходить усі `.cpp` файли в `src` та підпапках.

## Запуск
Після успішної компіляції:
```
./bookshop
```

## Модулі
- `user` — відображення меню користувача, базові дані
- `showBooks` — перегляд списку книг та детального меню
- `Basket` — робота з кошиком: додавання/видалення/відображення
- `productcard` — деталізація вибраного товару
